# LM Power Flow Validation

## 📌 Overview
This repository contains a MATLAB implementation and validation of a **Levenberg–Marquardt (LM)** based power flow method for robust analysis near voltage stability limits.

The project compares the classical **Newton–Raphson** method with the LM approach using the **IEEE 14-bus system**, demonstrating improved robustness of LM near the feasibility boundary.

---

## 🎯 Objectives
- Review a Levenberg–Marquardt based continuous power flow method
- Implement power flow solvers in MATLAB
- Analyze system behavior under increasing load conditions
- Compare Newton–Raphson and LM performance
- Identify voltage stability limits

---

## ⚙️ Tools and Environment
- MATLAB
- MATPOWER toolbox
- IEEE 14-bus test system
